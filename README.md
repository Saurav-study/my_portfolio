## 🛠️ The Tech Stack
Custom-built without heavy frameworks to ensure maximum performance and minimal payload.

* **Logic:** JavaScript (ES6+ Engine)
* **Styling:** CSS3 (Variable-driven, Grid & Flexbox)
* **Structure:** Semantic HTML5
* **Interactions:** Intersection Observer API & 3D Perspective Math

---

## ✨ Engineering Highlights

### 1. 3D Spatial Awareness
The "About" and "Tech Stack" containers utilize a **Dynamic Perspective Engine**. Using custom JavaScript math, the cards tilt in real-time based on cursor proximity, creating a tangible sense of depth.

### 2. Vertical Roadmap Architecture
Unlike standard project grids, this portfolio features a **Neon Spine Roadmap**.
- **The Spine:** A variable-gradient line that acts as a visual anchor.
- **Bi-Directional Nodes:** Projects alternate sides based on viewport width, connected by directional arrows for logical flow.

### 3. Interactive Profile Aura
The hero avatar features a **Neon Ripple Trigger**. Upon interaction (click or touch), a radial burst is generated through CSS keyframes, providing instant haptic-style visual feedback.

### 4. Cinematic Entrance
The name entry utilizes a **Blur-to-Clarity Slide**. By combining `cubic-bezier` timing with a horizontal translation and Gaussian blur, the header "ignites" upon page load.
